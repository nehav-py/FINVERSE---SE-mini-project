📌OVERVIEW

Finverse is a smart banking simulation web application designed to simplify personal finance management through intelligent automation and an engaging user experience. It integrates modern financial tools with interactive features to help users track expenses, manage budgets, and improve saving habits.

The platform introduces innovative concepts such as AI-powered card switching and a gamified savings module, making financial management both efficient and user-friendly.

As part of a Software Engineering major project, Finverse was designed following structured development practices. Various UML diagrams were created using StarUML to model and visualize the system architecture and workflows.
